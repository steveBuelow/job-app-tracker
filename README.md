# Job Application Pipeline Tracker

## In progress of being created!
A demo full-stack management web application built with Flask and SQLite.

## Features & Goals
* **Full-Stack CRUD Architecture:** Developed a management system to track the complete recruitment lifecycle (Wishlist, Applied, Interviewing, Offer, Rejected).

* **Relational Data Integrity:** Implemented a SQLite schema utilizing Foreign Key constraints to link job applications to unique, authenticated user profiles.

* **Asynchronous State Management:** Engineered backend logic to handle real-time status transitions via the Fetch API, allowing for dynamic UI updates without page refreshes.

* **Modern Professional UI:** Designed a "Corporate-Minimalist" interface focusing on CSS Flexbox and modern design systems to move beyond legacy table-based layouts.

* **Analytical Dashboard: (Goal)** A dedicated statistics view using SQL Aggregations to visualize application conversion rates and weekly "search velocity."

* **Document Association: (Goal)** Logic to link specific resume and cover letter versions to individual application entries via external URLs.
